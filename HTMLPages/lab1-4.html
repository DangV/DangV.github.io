Common HTML Tags

These are the most commonly used HTML tags and a short description of each.

Block Elements

Block elements take up their own line and take up the full width of the page.

body

This tag defines the documents body.

head

This tag defines information about the document.

html

This tag tells the browser that this in an HTML document. This is the container for all other elements.

div

This tag defines a division or sections in a document.

section

This tag definies sections in a document, such as chapters, headers, footers, or any other sections of the document.

header

This element defines a header for a document or section.

main

This tag specifies the main content of a document.

article

This tag specifies independent, self-contained content.

nav

This tag defines a set of navigation links.

footer

This tag defines a footer for a document or section.

p

This tag defines a paragraph.

aside

This tag defines some content aside from the content it is placed in.

li

This tag defines a list item. It is used in ordered list (ol), unordered lists (ul), and in menu lists (menu)

Inline Elements

Inline elements only take up as much space on a line as they need to. They do not start new lines.

a

This tag defines a hyperlink.

strong

This element makes text bold.

em

This tag makes text italicized.

img

This tag embeds an image on the page. You must use a src attribute to pick the image.

span

This tag is used to group inline-elements in a document.
